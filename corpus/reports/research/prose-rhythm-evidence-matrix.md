# 文章リズムに関する証拠表

## 判定ラベル

- **直接**: その主張を対象にした実験・分析がある
- **間接**: 関連する現象は確認されているが、その主張自体は検証していない
- **自前**: 主な根拠が本リポジトリのコーパス実験である
- **未確認**: 今回の調査では直接の根拠を確認できなかった
- **反証的**: 単純な一般化に反する結果や、強い条件依存性が確認された

書誌情報とリンクは、[文献レビュー](./prose-rhythm-literature-review.md) の「主要文献」にまとめた。

| 主張・実装 | 外部研究の状況 | 判定 | 扱い |
|---|---|---:|---|
| 文長は文体特徴である | Yule 1939、Sichel 1974、李・金 2019など | 直接 | 維持できる |
| 文長の並び順は散文リズムとして分析できる | Roberts 1996が自己相関を分析 | 直接 | 「分散」と「順序」を区別する |
| 文長の測り方はテキスト構造のproxyとして比較的頑健 | Marinho et al. 2018 | 直接 | 文字数・語数の違いは残るので日本語では再検証する |
| 文長分散が小さい文章は単調である | 読者の単調感を直接測った研究は確認できず | 未確認 | 「単調さ」ではなく「文長の均質さの疑い」とする |
| 文長分散が大きいほど読みやすい | 直接の支持なし | 未確認 | 品質規範にしない |
| LLM文は人間文より文長分布が均質 | 英語ニュースではMuñoz-Ortiz et al. 2024が支持 | 間接 | 言語・ジャンル・モデル限定と明記する |
| 日本語LLM文は文長分散が小さい | 自前452文書では人間mora CV中央値0.698、AI 0.431。ただしジャンル差が大きい | 自前 | 自前コーパスの範囲に限定する |
| `low_burstiness` はAIらしさを示す | 自前では差があるがCVの単調変換で同一信号。現行閾値の人間FP率32.4% | 自前 | 重複を明記し、AI判定ではなく補助的な編集ヒューリスティックにする |
| 文長の自己相関が高いと不自然 | 自前では人間中央値0.199、AI -0.053と仮説が逆。Roberts 1996も良否は判定しない | 反証的 | `high_length_autocorrelation` のAI的という解釈を外す |
| 読点は黙読時の韻律・構文処理に影響する | Steinhauer & Friederici 2001、Hirotani et al. 2006、Drury et al. 2016 | 直接 | 維持できるが、個数から良否は判定しない |
| 一文一義は理解を助ける | 吉田ほか 2002で解答時間の短縮 | 直接 | 読みやすさ原則の根拠にできる |
| 長い係り受けは読みにくい | 日本語では距離分布と解析有効性は確認。理解実験は未確認 | 間接 | 心理的負荷の断定には別の根拠が必要 |
| 読点の位置・間隔は書き手の特徴である | 金・樺島・村上 1993、岩崎 2018 | 直接 | 文体特徴として扱える |
| 文末表現は文体特徴である | 孟 2022など | 直接 | 維持できる |
| 体言止めの欠如はAIらしさの信号 | 直接の先行研究なし | 自前 | 自前コーパス限定のinfo信号にとどめる |
| 反復は不自然である | parallelismは意図的な修辞。機械識別も難しい | 反証的 | 反復を一律修正せずAI判断に残す現行方針が妥当 |
| 機能語・助詞・品詞系列は著者性を捉える | 金 2013、黄・金 2020 | 直接 | 文体分析には使える |
| 機能語・助詞・読点は日本語人間文とChatGPT文を分ける | Zaitsu & Jin 2023 | 直接 | 初期モデル・学術文限定。一般判定にしない |
| 語彙多様性が低いほどAIらしい | 研究によって方向が逆転。TTRは長さ依存 | 反証的 | 文書長制限と補助信号扱いを維持する |
| perplexityが低いほどAIらしい | モデル・言語・sampling依存。相対化した手法でも脆弱性あり | 反証的 | 本スキルの品質指標には採用しない |
| ジャンル別閾値が必要 | 文体・検出性能ともジャンル差が大きい | 直接 | 現行設計を支持する |
| 英語研究の閾値を日本語へ移せる | MULTITuDEは言語間一般化の不安定さを示す | 反証的 | 日本語コーパスで校正する |
| 表層特徴だけでAI生成を確定できる | M4、RAID、Sadasivan et al. 2023が汎化・改変への弱さを示す | 反証的 | lintを判定器にしない現行設計を支持する |

## 検出器ごとの暫定評価

| 検出器 | 研究上の位置づけ | 信頼してよい範囲 |
|---|---|---|
| `low_sentence_variance` | 文長分布という確立した文体特徴を使うが、「単調さ」への写像は未検証 | 文長が均質な箇所を見直す契機 |
| `low_burstiness` | `mora_cv` の単調変換で独立の情報を持たない。現行-0.24はFP 32.4%、-0.36ならFP 2.8%・AI検出57.5% | CVとの重複整理と閾値再校正が必要 |
| `high_length_autocorrelation` | 自前コーパスでは識別方向が仮説と逆で、現行閾値の発火も人間1.4%・AI0.5% | AI的な単調さの検出器としては削除候補 |
| `nominal_ending` | 文末は文体特徴。体言止め欠如とAIらしさの関係は自前 | 他のfindingと併用するinfo |
| `paragraph_lead_conjunction` | 接続詞・読点はレジスター差を持つ | ジャンル別の過剰使用の疑い |
| `uniform_paragraph_structure` | 人間・AIの段落長差を示す限定的研究はある | 自前データで校正した編集上の疑い |
| `repeated_sentence_lead` | 反復は文体・修辞特徴。意図的parallelismとの区別が必要 | 一律修正せず文脈判断 |
| `repeated_syntax_template` | 自動的な修辞構造判定は難しい | 連続する明白なテンプレの候補抽出 |
| `low_lexical_diversity_ttr` | TTRは長さ依存が強い | 十分長い文書で参考値のみ |
| `low_lexical_diversity_mtld` | TTRより長さに頑健だが、品質やAI性とは一致しない | 語彙反復の見直し候補 |
| `semantic_topic_flatness` | 文埋め込みによる文体分析は発展途上 | EXPERIMENTALのままが妥当 |

## 調査結果から導ける次の検証

優先順位が高いのは、検出器を増やすことではなく、現在の代理変数が読者の判断と対応するかを確かめることである。

1. 同じ内容について文長系列だけを操作した日本語文を作る
2. 読みやすさ、単調さ、自然さを別々に評定してもらう
3. 標準偏差、変動係数、隣接差、自己相関を別々に計算する
4. 評定との相関と、ジャンルによる交互作用を見る
5. 人間文／AI文のラベルは副次的に扱い、品質評価と混ぜない

この実験によって初めて、`low_sentence_variance` や `high_length_autocorrelation` が「統計的に差がある」だけでなく、読者が感じるリズムを捉えているかを検証できる。

コーパス上の識別力の比較は、[文長系列分析](./sentence-length-analysis.md) と
`corpus/experiments/rhythm/sentence_length_analysis.py` で実施した。次に残るのは読者評定との対応である。
